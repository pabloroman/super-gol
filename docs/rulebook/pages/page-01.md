**Super GOL** · Instrucciones

## *"Fútbol es fútbol"*
### M. Miljanic

## SUPER GOL.

Es un juego de naipes para dos jugadores, en el que se representa un partido de
fútbol. Para jugar, además de los naipes, únicamente necesitarás un par de dados
de 6 caras, el campo, el balón y mucha afición al fútbol.

Para convertirte en el mejor entrenador, te recomendamos que primero domines el
**JUEGO BÁSICO**. Después descubre todas las demás posibilidades que te ofrece el
**JUEGO AVANZADO**, teniendo en cuenta que puedes introducir las características y
reglas de esta modalidad UNA A UNA.

Elige bien tus jugadores, su posición en el campo, la estrategia de conjunto, los
cambios y demuestra que eres… **¡EL MEJOR ENTRENADOR!**

## EL CAMPO:

Tiene 30 casillas (6x5), por donde se desenvuelven los jugadores, más dos
habilitadas especialmente para los porteros. Existen zonas especiales, que son:

- **Zona de remate cercano (RM):** Es el lugar desde donde se puede chutar a gol
  y rematar de cabeza.
- **Zona de disparo lejano (DL):** Se puede chutar a gol desde fuera del área.
- **Zona de pases altos (PA):** Desde ahí se pueden realizar los pases altos.

### Diagrama del campo

Cuadrícula de **5 columnas × 6 filas** (las 30 casillas), con una casilla de
portería **arriba y otra abajo** — cada equipo ataca hacia la portería contraria.
Las zonas están rotuladas tal como aparecen en el escaneo:

```
                     ┌────────┐
                     │Portería│
                     └────────┘
   ┌────────┬────────┬────────┬────────┬────────┐
   │   PA   │   RM   │   RM   │   RM   │   PA   │
   ├────────┼────────┼────────┼────────┼────────┤
   │   PA   │ PA+DL  │   DL   │ PA+DL  │   PA   │
   ├────────┼────────┼────────┼────────┼────────┤
   │        │        │        │        │        │
   ├────────┼────────┼────────┼────────┼────────┤
   │        │        │        │        │        │
   ├────────┼────────┼────────┼────────┼────────┤
   │   PA   │ PA+DL  │   DL   │ PA+DL  │   PA   │
   ├────────┼────────┼────────┼────────┼────────┤
   │   PA   │   RM   │   RM   │   RM   │   PA   │
   └────────┴────────┴────────┴────────┴────────┘
                     ┌────────┐
                     │Portería│
                     └────────┘
```

Notas de lectura del escaneo:

- Las **columnas exteriores** (1 y 5) son **PA** en las cuatro filas con zona; las
  dos **filas centrales** no tienen zona especial (son casillas de juego normal).
- `PA+DL` son las casillas donde el escaneo apila **ambos** rótulos (PA sobre DL):
  cuentan como zona de pases altos **y** de disparo lejano a la vez.
- Las dos casillas de portería son las «dos habilitadas especialmente para los
  porteros» del texto, adicionales a las 30.
